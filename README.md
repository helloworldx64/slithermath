# כפל רויאל — Multiplication Royale

משחק דפדפן חינוכי בעברית לתרגול לוח הכפל, בסגנון משחקי אקשן מודרניים (סגנון slither.io).

שחקן פותח תיבות כפל, עונה על שאלות כפל, אוסף אבני בונוס, מפעיל טורבו ודש, והמשחק מתאים את הרמה לפי הצלחות וטעויות.

## איך משחקים
- **שולחן עבודה:** עכבר לניווט · קליק / רווח לדש · ענה נכון על כפל = טורבו
- **נייד:** ג'ויסטיק מגע בצד שמאל · כפתור ⚡ לדש בצד ימין

## תכונות
- 46 עיצובי נחש הנפתחים לפי רמה (מערכת XP והתקדמות)
- מנוע חישוב אדפטיבי — מתאים רמת קושי לפי דיוק ורצף
- 4 שרתים (אירופה / אמריקה / אסיה / ישראל)
- 24 בוטים בשלוש רמות קושי (קל / רגיל / קשה)
- צליל ומוזיקה מסונתזים (Web Audio, ללא קבצי נכסים)
- סטטיסטיקה מתמדת, חנות עיצובים, הגדרות, השהיה
- תמיכה מלאה במובייל, RTL, רספונסיבי
- אפקטים: חלקיקים, רעידת מסך, זוהר טורבו, עיניים עוקבות אחרי הסמן

## הרצה מקומית
פשוט פתח את `index.html` בדפדפן, או הרץ שרת סטטי:
```
python -m http.server 8000
```
